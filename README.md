# 🛒 Product Management System (CRUD)

A fully functional **CRUD** application to manage products efficiently. This project demonstrates how to Create, Read, Update, and Delete data using JavaScript.

## 🚀 Features
- **Add Products:** Quickly add new items to the list.
- **Search:** Real-time search for products by name.
- **Update:** Edit existing product details.
- **Delete:** Remove products one by one or clear all at once.
- **Validation:** Ensures all fields are filled before adding.
- **Persistence:** Uses **LocalStorage** to keep your data even after refreshing!

## 🛠️ Built With
- **HTML5 & CSS3**
- **Bootstrap 5** (For responsive design)
- **Vanilla JavaScript** (ES6+)
- **Google Fonts**
